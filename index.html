Step 01
Prerequisites

Claw Code's Rust core requires a Rust toolchain. The Python workspace works standalone. Install what you need for your chosen track.

For Rust track (recommended)

shell
# Install Rust via rustup curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
shell
# Reload shell, then verify source $HOME/.cargo/env rustc --version # rustc 1.78+ cargo --version # cargo 1.78+
🪟

On Windows, install Rust using winget (Windows 11/10) or the official GUI installer from rustup.rs.

powershell
# Option A — winget (recommended) winget install Rustlang.Rustup
powershell
# Option B — PowerShell installer Invoke-WebRequest -Uri "https://win.rustup.rs/x86_64" -OutFile rustup-init.exe .\rustup-init.exe
powershell
# Verify install (new terminal) rustc --version # rustc 1.78+ cargo --version # cargo 1.78+

For Python track

shell
python3 --version # needs Python 3.9+
powershell
# Install Python 3.12 via winget winget install Python.Python.3.12 python --version # needs Python 3.9+
💡

On Windows, run all commands in PowerShell (not CMD). After installing Rust, restart your terminal so cargo appears on your PATH.

4
Step 04
Build

Build the Rust workspace or set up the Python environment depending on your chosen track.

🦀 Rust Track

shell
cd rust cargo build --release # This compiles the full workspace: # crates/api-client — API + OAuth + streaming # crates/runtime — session, MCP orchestration # crates/tools — tool manifest & execution # crates/commands — slash commands # crates/plugins — plugin pipeline # crates/claw-cli — interactive REPL binary # crates/server — HTTP/SSE server (axum) # crates/lsp — LSP client integration
⏱️

First build takes 2–5 minutes as Cargo downloads and compiles dependencies. Subsequent builds are fast thanks to incremental compilation.

🐍 Python Track

shell
# From repo root — no build step needed! python3 -m unittest discover -s tests -v
✨

Python track requires zero compilation. All tests should pass out of the box with Python 3.9+.

5
Step 05
Run & Verify

Fire up Claw Code and verify everything is working. Below are all the commands you'll use regularly.

🦀 Rust — Run the binary

shell
# Launch interactive REPL ./target/release/claw-cli
🚀

The binary is at rust/target/release/claw-cli relative to the repo root. You can also cargo run --release from inside the rust/ directory during development.

🐍 Python — Core commands

shell
# Render porting summary python3 -m src.main summary # Print current workspace manifest python3 -m src.main manifest # List Python modules (limit 16) python3 -m src.main subsystems --limit 16 # Inspect command inventory python3 -m src.main commands --limit 10 # Inspect tool inventory python3 -m src.main tools --limit 10 # Run parity audit (needs local archive) python3 -m src.main parity-audit

Verification checklist

  • ✓ Rust build completes without errors
  • ✓ python3 -m unittest discover — all tests pass
  • ✓ src.main summary outputs porting overview
  • ✓ claw-cli REPL launches and accepts input
